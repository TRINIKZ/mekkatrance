GIMP is the tool I believe I have the most mastery over; I've been using it since 2015. Since then, I've developed my skills in it by practicing to create my own wallpapers. Blender

I began using Blender as a tool to assist me in creating avatars in 2019, and this lasted for about a year, until I started doing renders. I enjoyed creating abstract landscapes, but I wanted to level up, and that's when I truly began modeling to be able to create absolutely anything I wanted.
